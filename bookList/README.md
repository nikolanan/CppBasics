Firstly, an integer corresponding to the number of books is entered. Then, a title and a year of release for each book. In the end, the program displays all of the data for each book.

Sample input: 

2 

The Great Gatsby 

1925 

The catcher in the rye 

1954

Sample output:

Book number 1

The Great Gatsby 

1925 

Book number 2

The catcher in the rye 

1954
